
#                                                      Quan-Ly-Thoi-Gian-Su-Dung-androidTV

## Dành cho các thiết bị chạy androidTV từ phiên bản 11 trở lên.

## Chức năng
  + Khóa app theo thời gian do người dùng ấn định.
  + Hiển thị thời gian sử dụng của từng ứng dụng trong 24h.
  
  
## Powered by
-  Java
-  XML
-  Android Studio
